# Blood Donation Platform

## Purpose

A full-stack blood donation platform connecting donors, recipients, volunteers, and admins.

## Live Site

Coming Soon

## Key Features

- Role-based dashboard (Admin, Donor, Volunteer)
- Secure authentication with JWT
- Blood donation request management
- Donor search by blood group & location
- Funding system with Stripe

## Technology Stack

Frontend:

- React, React Router, Tailwind CSS, DaisyUI, Firebase, Axios, TanStack Query

Backend:

- Node.js, Express.js
- MongoDB
- JWT, Stripe
